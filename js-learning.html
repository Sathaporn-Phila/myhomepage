Where to place script

script ของ javascript มีลักษณะดังนี้

<script> document.getElementById("demo").innerHTML = "My First JavaScript"; </script> โดย script นั้นสามารถเขียนได้แบบ Internal หรือ External



Internal

การเขียนแบบ Internal คือการเขียนโค้ด javascript ไว้ในไฟล์ html เขียนได้ในบริเวณ <head> หรือ <body>ตัวอย่างเช่น

<head> <script> function myfunction1(){ alert("Hello Sun"); } </script> </head> <body> <script> function myfunction2(){ alert("Hello Moon"); } </script> </body>


External

การเขียนแบบ External คือการเขียนโค้ด javascript แยกกับไฟล์ html แต่สามารถเรียกใช้ function ผ่าน javascript ได้โดย <script src="..."> ตัวอย่างเช่น

<head> <script src="../javascript/ball.js"></script> </head>

การเขียนจาก External ไม่ต้องมีโค้ด javascript ใน <script> เพราะจะถูกเรียกคำสั่งจากด้านนอกแทนอย่างเช่นใน ไฟล์ ball.js มี function myfunction3 ซึ่งจะ alert อะไรบางอย่างบนหน้าจอ



Output

Output ใน Javascript สามารถแสดงผลลัพธ์ได้ 4 หนทาง


ใช้ innerHTML

เพื่อการเข้าถึง HTML element จำเป็นต้องใช้ id คือ document.getElementById(id) ตัวอย่างเช่น

<p id="demo"></p> <script> document.getElementById("demo").innerHTML = "Hello World"; </script>

เมื่อทำการกด Show element ตัว HTML content จะมีการเปลี่ยนแปลงให้โชว์ Hello World



ใช้ document.write()

การใช้ document.write() เป็นการแสดงผลลัพธ์บน HTML ไม่ควรใช้หลังจากข้อมูลบนเว็บเพจ show บนเว็บเพจ เพราะจะทำให้ข้อมูลบนเว็บเพจสูญหาย ตัวอย่างโค้ด document.write() เช่น

<script>document.write("Hello Sun");</script>

ใช้ alert()

การใช้ alert(text) เป็นการไว้ pop-up ข้อความออกมา ตัวอย่างเช่น

<script> alert("Hello Moon"); </script>

เมื่อทำการกดปุ่ม จะมี pop-up ข้อความบางอย่างออกมา



ใช้ console.log()

การใช้ console.log() เป็นการใช้ debugged code บน HTML หรือ script ตัวอย่างเช่น

<script> console.log("Surprising"); </script>

ลองกดปุ่ม F12 จะเห็นคำว่า Surprising
