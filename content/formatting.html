Formatting

Formatting ใน HTML เป็นการเปลี่ยนแปลง text ใน HTML เพื่อให้มีความพิเศษมากยิ่งขึ้น ตัวอย่างเช่น

  • <b> เป็นตัวที่ทำให้ text หนาขึ้น ตัวอย่างการใช้

    <b>Hello World</b> = Hello World

  • <strong> เป็นตัวที่ทำให้ text ถูกเน้นให้มีความสำคัญ ตัวอย่างการใช้

    <strong>Hello World</strong> = Hello World

  • <i> เป็นตัวที่ทำให้ text ถูกเน้นให้เอียง ตัวอย่างการใช้

    <i>Hello World</i> = Hello World

  • <em> เป็นตัวที่ทำให้ text ถูกเน้นให้เอียง เหมือนกับ <i>

  • <small> เป็นตัวที่ทำให้ text ถูกทำให้เล็กลง ตัวอย่างการใช้

    <small>Hello World</small> = Hello World

  • <mark> เป็นตัวที่ทำให้ text ถูก highlighted ตัวอย่างการใช้

    <mark>Hello World</mark> = Hello World

  • <del> เป็นตัวที่ทำให้ text ถูกขีดฆ่า หรือ strikeout ตัวอย่างการใช้

    <del>Hello World</del> = Hello World

  • <ins> เป็นตัวที่ทำให้ text ถูกเพิ่มไปใน document โดย Browser มองเห็นเป็นการขีดเส้นใต้ ตัวอย่างการใช้

    <ins>Hello World</ins> = Hello World

  • <sub> เป็นตัวที่ทำให้ text ถูกห้อยอยู่ด้านล่าง ตัวอย่างการใช้

    H<sub>2</sub> = H2

  • <sup> เป็นตัวที่ทำให้ text ถูกห้อยอยู่ด้านบน ตัวอย่างการใช้

    H<sup>2</sup> = H2
