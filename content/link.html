Links

link ใน HTML คือ การกำหนด element ที่สามารถไปอีกเว็บไซต์ได้หรืออีกไฟล์นึงใน link เดียวกันได้โดยใช้แท็ก <a>

การระบุเป้าหมาย link

การระบุเป้าหมาย link สามารถทำได้โดยกำหนด attributes (href) ใน <a> ตัวอย่างโค้ด <a href="element.html" >Elements page</a>     Elements page

การระบุเป้าหมาย target

การระบุ target สามารถทำให้เป้าหมาย link ไปที่ไหน เช่น frame หลัก หรือ iframe หรือแท็บใหม่ เป็นต้น ตัวอย่างโค้ด <a href="www.google.com" target="_blank">Google แท็บใหม่</a>     Google แท็บใหม่
