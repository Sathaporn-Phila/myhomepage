Head

Head เป็น metadata ที่อยู่ระหว่าง <html> กับ <body> ซึ่ง metadata นั้นจะไม่แสดงออกมาทางหน้าจอ Head ประกอบด้วย 6 อย่าง คือ

  1. title เป็นหัวข้อชื่อ tab ซึ่งอยู่บน browser toolbar สิ่งนี้จำเป็นต่อ search engine ลักษณะโค้ดดังนี้ <!DOCTYPE html> <html> <head> <title>A Meaningful Page Title</title> </head> <body> The content of the document...... </body> </html>
  2. style เป็นการกำหนด CSS ใน tag มีลักษณะการเขียนโค้ดดังนี้ <!DOCTYPE html> <html> <head> <title>A Meaningful Page Title</title> <style> body {background-color: powderblue;} h1 {color: red;} p {color: blue;} </style> </head> <body> The content of the document...... </body> </html>
  3. link เป็นการเรียก source จาก external ตัวอย่างโค้ด เช่น <link rel="stylesheet" href="mystyle.css>
  4. meta เป็น element พิเศษที่ใช้เรียกตัวอักษรพิเศษ คำอธิบายเพจ keyword ผู้แต่งเว็บเพจ viewport (viewport คือ การปรับ scale ตาม device) เป็นต้น ตัวอย่างโค้ด เช่น <meta name="viewport" content="width=device-width, initial-scale=1.0">
  5. script เป็น element ที่ใช้เรียกภาษา javascript ตัวอย่างเช่น <script> function myFunction() { document.getElementById("demo").innerHTML = "Hello JavaScript!"; } </script>
  6. base เป็น element ที่ base url หรือ target ไปที่ relative url ตัวอย่างโค้ดเช่น <head> <base href="https://www.w3schools.com/" target="_blank"> </head> <body> <a href="tags/tag_base.asp">HTML base Tag</a> </body> ในที่นี้จะเรียก www.w3schools.com ตามด้วย tags/tag_base.asp แล้วขึ้น tab ใหม่เป็น www.w3schools.com/tags/tag_base.asp
