Lists

Lists ใน HTML เป็นการ set item ให้อยู่เป็นกลุ่มๆ มี List อยู่ 3 ประเภท

Unorder list

เป็นกลุ่มข้อมูลที่ไม่จำเป็นต้องเรียงลำดับ มีลักษณะการโค้ดดังนี้

<ul> <li>cats</li> <li>dogs</li> <li>elephants</li> <li>fish</li> </ul> <ul> คือ unorder list ส่วน <li> คือ list ของข้อความ โดยมีผลลัพธ์คือ
  • cats
  • dogs
  • elephants
  • fish

Order list

เป็นกลุ่มข้อมูลที่ต้องเรียงลำดับ มีลักษณะการโค้ดดังนี้

<ol> <li>cats</li> <li>dogs</li> <li>elephants</li> <li>fish</li> </ol> <ol> คือ order list ส่วน <li> คือ list ของข้อความ โดยมีผลลัพธ์คือ
  1. cats
  2. dogs
  3. elephants
  4. fish

Description list

เป็นกลุ่มข้อมูลที่ต้องการอธิบาย item มีลักษณะการโค้ดดังนี้

<dl> <dt>cats</dt> <dd>สัตว์ร้องเหมียว</dd> <dt>dogs</dt> <dd>สัตว์ร้องโฮ่ง</dd> <dt>elephants</dt> <dd>สัตว์มีงา</dd> <dt>fish</dt> <dd>สัตว์ที่ว่ายน้ำได้</dd> </dl> <dl> คือ description list ส่วน <dt> คือ item ใน description และอันสุดท้ายคือ โดยมีผลลัพธ์คือ <dd> คือคำอธิบาย item
cats
สัตว์ร้องเหมียว
dogs
สัตว์ร้องโฮ่ง
elephants
สัตว์มีงา
fish
สัตว์ที่สามารถว่ายน้ำได้
