Styles

style คือ เป็น attributes ของ element และ style สามารถเขียนอีกรูปหนึ่งในภาษา CSS (Cascading Style stylesheet) ซึ่ง style นั้นเอาไว้ใช้ตกแต่งให้สวยงาม ตัวอย่างเช่น

  1. <p style="color:blue;">ฉันเป็นคนไทย</p> style ตัวนี้เอาไว้ใช้เปลี่ยนสีตัวอีกษร

    ฉันเป็นคนไทย
