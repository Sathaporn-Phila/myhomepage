Classes

class เป็น attributes ของ elements ใช้สำหรับระบุ element เฉพาะ นอกจากนี้ยังใช้ร่วมกับ CSS หรือ Javascripts ตัวอย่างเช่น

CSS <style> .city {color: red;} </style> <p class="city">Bangkok</p> Javascripts <button onclick="myfunction()">Show Elements</button> <p class="city"></p> <script> function myfunction() { document.getElementsByClassName("city")[0].innerHTML = "Bangkok"; } </script>
