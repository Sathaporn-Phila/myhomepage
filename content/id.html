Id

Id คือ attribue ที่มีไว้บ่งบอกความเป็นเฉพาะเจาะจงของ element เหมือนกับ class แต่สามารถมีได้ id เดียว สามารถใช้ร่วมกับ CSS หรือ Javascripts

CSS <style> #city {color: red;} </style> <p id="city">Bangkok</p> Javascripts <button onclick="myfunction()">Show Elements</button> <p id="city"></p> <script> function myfunction() { document.getElementById("city").innerHTML = "Bangkok"; } </script>
