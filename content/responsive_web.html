Responsive web design

Responsive web design คือ การออกแบบเว็บให้มีความเหมาะสมกับทุก device ไม่ว่าจะเป็นโทรศัพท์หรือโน้ตบุ้คหรือจอต่างๆที่มีขนาดความกว้างและยาวไม่เท่ากัน โดยการออกแบบ Responsive web design สามารถใช้ได้ทั้ง HTML และ CSS เพื่อปรับให้เว็บไซด์ของเราปรับขนาดได้อัตโนมัติ

ตัวอย่างโค้ด HTML ที่เป็น responsive ให้เข้ากับ device ต่างๆ เช่น

<head> <meta name="viewport" content="width=device-width, initial-scale=1.0"> </head>

ต่อมาจะกล่าวถึง CSS ซึ่งจะมีหัวข้อต่อไปนี้

Responsive Image

Responsive Image คือ ขนาดของภาพที่ปรับตามขนาดของเว็บไซต์ โดยสามารถทำได้ในรูปแบบของ CSS ตัวอย่างเช่น

ใช้ขนาดของ width ที่เป็น %

                
                .img {
                    width : 50px
                }

หรือจะใช้ตัวที่เป็น max-width กรณีขนาดเท่ากับขนาดของเว็บไซต์ซึ่งสามารถกำหนดอัตราส่วนเป็น % ตัวอย่างเช่น

<img src="../img/my_profile.jpg" style="max-width: 100%;height: auto;">

ไม่ว่าจะปรับขนาดเว็บไซต์ขนาดเท่าไหร่ก็ตาม รูปภาพก็ยังโชว์เต็มรูปภาพ

นอกจากนี้ยังมีคำสั่ง <picture> ที่ช่วยให้มีหลายรูปภาพในรูปเดียวโดยประยุกต์เข้ากับ attributes media ตัวอย่างเช่น

<picture> <source srcset="../img/kiwi.jpg" media="(max-width:600px)"> <source srcset="../img/waterbear.jpg" media="(max-width:1500px)"> <source srcset="../img/my_profile.jpg"> <img src="../img/kiwi.jpg" style="width:50%;height: auto;"> </picture>

เมื่อถึง width ของเว็บไซต์ระดับหนึ่งรูปภาพก็เปลี่ยนไป


Responsive Text

การทำให้ text ปรับตามขนาดเว็บไซต์ สามารถทำได้โดย set font size เป็นหน่วย vw โดยหน่วยนี่มีชื่อเรียกเต็มว่า viewport width ตัวอย่างโค้ดเช่น

<h5 style="font-size:4vw;">Hello world</h5>

เมื่อลองขยายหรือลดขนาดเว็บไซต์ ตัวอักษรจะขนาดเป็น 4% ของความกว้างของเว็บไซต์ตลอดเวลา

Hello world
