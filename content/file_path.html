File Path

File Path สามารถช่วยระบุแหล่งที่อยู่ของไฟล์ที่ต้องใช้ใน webpage จะถูกใช้เมื่อต้องการเชื่อมต่อกับไฟล์ภายนอก เช่น

  • webpage
  • รูปภาพ
  • CSS file
  • javascript file
การเชื่อมต่อกับไฟล์ภายนอก แบ่งออกเป็น 2 ประเภท
  • Absolute file path คือ แหล่งที่อยู่ของไฟล์แบบ full url ตัวอย่างเช่น <img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">
  • Relative file path คือ แหล่งที่อยู่ของไฟล์เทียบกับหน้าปัจจุบัน ตัวอย่างเช่น <img src="/images/picture.jpg" alt="Mountain">
