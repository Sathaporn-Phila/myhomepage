HTML มีบางสิ่งที่เอาไว้เรียกส่วนเฉพาะของ webpage มีดังนี้
  • <header> เป็นส่วนที่ไว้เรียก topic ของ section นั้นๆ
  • <nav> เป็นส่วนที่ไว้กลุ่ม link target
  • <section> เป็นส่วนที่ไว้เรียกกลุ่มข้อมูลใน webpage
  • <article> เป็นส่วนที่ไว้เรียกกลุ่มที่มีเนื้อหาเฉพาะตนเอง ไม่เกี่ยวกับ section อื่น
  • <aside> เป็นส่วนที่ไว้เรียกกลุ่มข้อมูลแถบข้าง
  • <footer> เป็นส่วนที่ไว้เรียกกลุ่ม footer
  • <details> เป็นส่วนที่ไว้เรียกการอธิบายของข้อมูล
  • <summary> เป็นส่วนที่ไว้เป็น topic ของ <details>
นอกจากนี้ Layout ยังสามารถถูกจัดการได้ด้วยภาษา CSS เช่น flex grid หรือยังเป็น Bootstrap เป็นต้น
