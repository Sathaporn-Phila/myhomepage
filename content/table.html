Table

Table ใน HTML เปรียบเสมือนการสร้างตารางใน web page มี tag ที่ต้องใช้ดังนี้

  • <table> เป็นการสร้างโครงตาราง
  • <tr> เป็นการสร้าง row 1 row
  • <th> เป็นการกำหนด data ในตารางให้มีลักษณะเป็น heading
  • <td> เป็นการกำหนด data ในตาราง
โดยมีวิธีการเขียนโค้ดดังนี้ <table> <tr> <th>File name</th> <th>File extension</th> </tr> <tr> <td>Myname.py</td> <td>py</td> </tr> <tr> <td>element.html</td> <td>html</td> </tr> </table>
File name File extension
Myname.py py
element.html html

การทำให้ข้อมูลในตารางกินพื้นที่มากกว่า 1 ช่องในแนวนอน สามารถทำได้โดยเพิ่ม attribute คือ rowspan ตัวอย่างเช่น

<table> <tr> <th rowspan="2">File name</th> </tr> <tr> <td>Myname.py</td> <td>py</td> </tr> <tr> <td>element.html</td> <td>html</td> </tr> </table>

บริเวณทีเป็น <th>File name</th> จะกินพื้นที่ไป 2 ช่องแนวนอน

การทำให้ข้อมูลในตารางกินพื้นที่มากกว่า 1 ช่องในแนวตั้ง สามารถทำได้โดยเพิ่ม attribute คือ colspan ตัวอย่างเช่น

<table> <tr> <th>File name</th> <th>File name</th> </tr> <tr> <td colspan="2">Myname.py</td> <td colspan="2">py</td> </tr> </table>
