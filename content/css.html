CSS

CSS คือ ภาษาที่ไว้ใช้ร่วมกับ HTML เอาไว้ใช้ตกแต่ง web page มีวิธีเขียน 3 แบบคือ

  • Inline เขียนตรง element แท็กเปิดใน style attribute ตัวอย่างเช่น <h1 style="color: red">I'm a dark eyes</h1> ผลลัพธ์ :

    I'm a dark eyes

  • Internal เขียนตรง head มีลักษณะการเขียนดังนี้ <head> <style> h1 {color: red}; </style> </head>
  • External คือ การเขียนภาษา CSS ไว้อีกไฟล์นึงแล้วให้ Link กับ HTML บริเวณ head ตัวอย่างเช่น color.css File h1 {color: red;} myfile.html File <head> <link rel="stylesheet" href="color.css" type="text/css"> </head>
